## 2024-06-28 | [Architectural Audit] | Insight: DaisyUI modal backdrops hide a visually invisible button that acts as the click-away close trigger, causing screen readers to announce unlabelled buttons. Agent status lists and command descriptions lacked structural safety. | Protocol: Always explicitly assign aria-label="Close dialog" to modal backdrop buttons, guarantee deterministic 4-state logic (Loading, Error, Empty, Success) for all async collections, and mandate explicit id and htmlFor bindings on all form inputs.
