## 2024-05-24 - When updating DOM structure, ensure you update associated Playwright E2E locators
**Learning:** Adding a wrapper `<div class="group relative">` around a `<pre>` element for hovering buttons breaks tests that target `.mockup-code pre.text-error`. Tests may need a slightly broader locator strategy like `.mockup-code pre.text-error, .mockup-code .text-error` to adapt seamlessly without complete test rewrites.
**Action:** When making DOM structure changes, ensure associated CSS locators in E2E tests are updated to target classes directly rather than strict parent-child structures if possible.
